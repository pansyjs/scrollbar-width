<h1 align="center">@pansy/scrollbar-width</h1>

<div align="center">
获取浏览器滚动条宽度
</div>

## 🏗 安装

```sh
# npm
npm i @pansy/scrollbar-width

# yarn
yarn add @pansy/scrollbar-width

# pnpm
pnpm i @pansy/scrollbar-width
```

## 使用

```ts
import { scrollbarWidth } from '@pansy/scrollbar-width';

scrollbarWidth();
```
